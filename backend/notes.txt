About JWT Info:
 jwt.sign(payload, secret, options)
        payload = data stored inside the token { user_id, user_role }
        JWT_SECRET = a secret key from your .env file used to sign/verify the token
        expiresIn: '7d' = token expires in 7 days
        This token is like a key — user sends it with every request
